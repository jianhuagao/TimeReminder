# ⏰ TimeReminder（AutoHotkey v2 桌面时间提醒小组件）

一个基于 **AutoHotkey v2** 的轻量级桌面时间提醒工具，采用小组件风格浮窗，支持多时间点提醒、倒计时显示、勿扰模式、开机启动、托盘菜单等功能。

> 适合：会议提醒、下班提醒、番茄时间节点、课程时间、日程节点等。

---

## ✨ 功能特点

### 🕒 多时间点提醒

- 支持多个提醒时间
- 每日循环（跨天自动刷新）
- 自动选择下一个最近的目标时间

---

### ⏳ 倒计时小组件

- 常驻桌面（可关闭）
- 显示：
  提醒时间：HH:MM
  倒计时 mm:ss

- 到点显示「已到点」
- 默认只在最后 2 分钟显示（可改为常驻）

---

### 🔔 多阶段提前提醒

默认提前：

- 5 分钟
- 2 分钟
- 1 分钟

使用系统托盘气泡通知。

---

### 🧩 小组件交互

| 操作           | 功能     |
| -------------- | -------- |
| 点击标题栏     | 拖动窗口 |
| 点击内容区     | 打开配置 |
| 点击右下角 `⋯` | 打开菜单 |
| 托盘图标       | 同样菜单 |

---

### 🧠 菜单功能

- 打开配置
- 显示 / 隐藏小组件
- 暂停 / 继续提醒
- 勿扰模式
- 常驻浮窗开关
- 开机启动
- 立即刷新
- 退出

---

### 🌙 勿扰与智能静音

- 勿扰模式：完全不弹提醒
- 全屏静音：游戏 / 视频时不打扰

---

### 🎨 UI 特性

- 无边框小组件风格
- 圆角窗口（WinAPI 实现，稳定）
- 半透明（可调）
- 深浅分区设计
- 位置记忆
- 始终置顶

---

## ⌨️ 输入格式（非常简单）

在配置窗口输入目标时间：
1717 1818 2030

规则：

- 使用 **空格分隔**
- 每组 **4 位数字**
- 格式：`HHMM`
- 自动转换为 `HH:MM`

---

### ✅ 示例

| 输入 | 实际时间 |
| ---- | -------- |
| 1717 | 17:17    |
| 0830 | 08:30    |
| 2359 | 23:59    |

---

### ✔ 兼容格式（会自动清洗）

17:17, 18:18
17:17 18:18
17:17;18:18

---

## ⚙️ 配置项说明

| 项目       | 说明              |
| ---------- | ----------------- |
| 常驻浮窗   | 始终显示倒计时    |
| 透明度     | 0–255             |
| 圆角半径   | 0–80              |
| 勿扰模式   | 禁止弹提醒        |
| 全屏静音   | 游戏/视频时不提醒 |
| 开机启动   | 自动随系统启动    |
| 启动弹配置 | 启动时打开设置    |

## 📦 安装与运行

### 1️⃣ 安装 AutoHotkey v2

下载：

👉 https://www.autohotkey.com/

⚠️ 必须为 **v2.x**

---

### 2️⃣ 运行脚本

双击 `.ahk` 文件即可。

---

### 3️⃣ 开机启动（可选）

在配置中勾选即可自动创建启动项。

---

## 📁 配置文件

脚本目录下会生成：
TimeReminder.ini

用于保存：

- 提醒时间
- UI 设置
- 小组件位置
- 开机启动状态
- 勿扰模式等

删除该文件可恢复默认设置。

---

## 🖱️ 小组件操作说明

### 拖动窗口

点击顶部深色区域拖动。

---

### 打开配置

点击中间区域。

---

### 打开菜单

点击右下角 `⋯`

---

### 隐藏小组件

菜单中选择「隐藏小组件」

## 🔄 工作逻辑

1. 每秒检测当前时间
2. 计算下一个目标时间
3. 触发提前提醒
4. 显示倒计时
5. 到点提醒
6. 自动切换下一个目标
