# TimeReminder - 时间提醒工具

TimeReminder 是一个基于 AutoHotkey v2.0 开发的轻量级时间提醒工具，用于在指定时间前发送多阶段提醒通知，并在最后2分钟显示倒计时悬浮窗。

## 功能特性

- ⏰ **多阶段提醒**：在目标时间前 5 分钟、2 分钟和 1 分钟分别发送提醒
- 📊 **倒计时悬浮窗**：在最后 2 分钟显示置顶倒计时窗口
- ⚙️ **配置灵活**：通过 ini 文件自定义提醒时间
- 📁 **自动配置**：首次运行自动创建配置文件
- 🪶 **轻量高效**：占用资源少，后台静默运行

## 安装与运行

### 前置要求

- 安装 [AutoHotkey v2.0](https://www.autohotkey.com/download/ahk-v2.exe) 或更高版本

### 安装步骤

1. 下载 `TimeReminder.ahk` 文件到本地
2. 双击运行该文件，程序会在后台启动并创建配置文件
3. 程序将在系统托盘运行，等待触发提醒

## 配置说明

程序首次运行时会在同目录下创建 `TimeReminder.ini` 配置文件，默认配置为：

```ini
[Time]
TargetHour=11
TargetMinute=0
```

### 配置选项

- **TargetHour**：目标小时（24小时制，0-23）
- **TargetMinute**：目标分钟（0-59）

### 修改配置

1. 关闭正在运行的 TimeReminder 程序
2. 编辑 `TimeReminder.ini` 文件，修改目标时间
3. 重新运行 `TimeReminder.ahk` 应用新配置

## 提醒机制

1. **5分钟提醒**：当距离目标时间还有5分钟时，发送系统托盘通知
2. **2分钟提醒**：当距离目标时间还有2分钟时，发送系统托盘通知并显示倒计时窗口
3. **1分钟提醒**：当距离目标时间还有1分钟时，发送系统托盘通知
4. **倒计时窗口**：在最后2分钟内持续显示，位于屏幕右下角，显示剩余分:秒

## 技术实现

- 使用 AutoHotkey v2.0 的 `Persistent` 模式保持后台运行
- 通过 `SetTimer` 每秒检查当前时间与目标时间的差距
- 利用 `TrayTip` 发送系统托盘通知
- 使用 `Gui` 创建倒计时悬浮窗口
- 通过 `IniRead` 和 `IniWrite` 读写配置文件
